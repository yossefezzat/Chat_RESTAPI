## README

Ruby on Rails Chat-RestApi running on Docker using mysql, redis, sidekiq, elasticsearch.

### Environment

* Ruby v2.7
* Rails v5
* Elasticsearch 7.17
* Mysql 5.7
* Redis 5.0
* sideKiq 

### Requirements

* Docker
* Docker compose

### Run

```bash
$ docker compose up -d --build
```
